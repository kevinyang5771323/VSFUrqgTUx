# 项目简介

基于Spring Boot的环境检测系统是一款旨在为用户提供环境信息监测服务的软件平台。

## 核心功能
- 展示热门景点
  - 获取并展示当前热门或推荐的景点列表
  - 链接到用户登录/注册页面
  - 根据用户偏好或当前热门推荐景区信息
  - 展示与旅游相关的商品推荐
- 景点详情页
  - 展示景点的详细信息，包括名称、描述、位置、评分等
  - 提供预订按钮，用户可以点击进行预订
  - 展示与该景点相关的商品推荐
- 用户登录/注册页
  - 提供用户登录表单，包含用户名和密码输入框
  - 提供用户注册表单，包含用户名、密码、邮箱等输入框

## 适用场景
- 旅游行业
- 城市规划
- 环境监测

## 技术特点
- 采用Spring Boot框架，易于开发和部署
- 支持前后端分离，提高开发效率和用户体验
- 数据库存储结构合理，保证数据安全性
